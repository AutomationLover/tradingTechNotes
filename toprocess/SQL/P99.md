- PostgreSQL / MySQL 8 / Oracle：用 ‎`percentile_disc(0.99) WITHIN GROUP (ORDER BY x)`。
- Hive / Presto / Trino / ClickHouse：用 ‎`approx_percentile(x, 0.99)` 等近似函数。
- 老数据库 / 无分位函数：用窗口函数排序 + ‎`ROW_NUMBER()` + ‎`COUNT()` 自己取第 ‎`ceil(0.99 * N)` 条记录。

```sql
WITH ordered AS (
  SELECT
    latency_us,
    ROW_NUMBER() OVER (ORDER BY latency_us) AS rn,
    COUNT(*)     OVER ()                     AS total
  FROM latency_samples
)
SELECT latency_us AS p99_latency_us
FROM ordered
WHERE rn = CEIL(0.99 * total);

```
